Atelier Strudel

Cet atelier a pour but de vous guider pas à pas dans la création de musique algorithmique en temps réel, directement dans le navigateur au travers de l'outil Strudel.

L'atelier est ouvert à tous·tes, pas besoin de connaissances en programmation pour devenir des livecodeur·euses.

Pas d'installation nécessaire. Tapez du code et appuyez sur Ctrl+Entrée pour jouer, et Ctrl+. pour arrêter.

Qu'est-ce que Strudel ?

Strudel est une implémentation JavaScript du langage de patterns musicaux Tidal Cycles, créé en 2022 par Alex McLean et Felix Roos.

Contrairement à Tidal Cycles (language écrit en Haskell qui nécessite le moteur audio SuperCollider), Strudel fonctionne directement dans le navigateur sans aucune installation.

Modifiers Aléatoires

Ces fonctions introduisent du hasard dans le pattern :

FonctionDescription

rarely(fn)Applique fn avec probabilité faible (~25 %)
almostNever(fn)Applique fn très rarement

irand(n)Entier aléatoire de 0 à n−1
--> degradeBy — Ouvrir dans Strudel --> sometimes — Ouvrir dans Strudel --> rarely + often — Ouvrir dans Strudel

Modifiers Conditionnels

mask(pattern)Filtre les événements selon un masque binaire (0/1)

Visualisations et interfaces graphiques

Strudel propose plusieurs visualisations qui s'attachent à un pattern pour en afficher le contenu en temps réel.

Fonction Description
.slider()Crée un curseur pour contrôler dynamiquement la valeur d'un paramètre
.pianoroll()Piano roll flottant : affiche les hauteurs dans le temps dans une fenêtre séparée
.spiral()Visualisation circulaire du pattern dans une fenêtre séparée
.scope()Visualisation oscilloscope
._pianoroll()Piano roll inline, directement dans l'éditeur
._punchcard()Grille rythmique inline (punch card)
._spiral()Spirale inline, directement dans l'éditeur
._scope()Visualisation oscilloscope inline, directement dans l'éditeur
// Piano roll flottant
note("c3 e3 g3 b3").pianoroll()

// Spirale inline
note("c3 [e3 g3] b3")._spiral()

// Slider pour contrôler le gain en direct
sound("bd sd").gain(slider(0.8, 0, 1))

Souris

Strudel expose deux fonctions pour utiliser la position de la souris comme source de contrôle :

Fonction Description
mousexPosition horizontale de la souris (valeur entre 0 et 1)
mouseyPosition verticale de la souris (valeur entre 0 et 1)
// Le cutoff du filtre suit la position horizontale de la souris
sound("bd*4 sd*2").cutoff(mousex.range(200, 4000))

// Le gain suit la position verticale (haut = fort)
note("c3 e3 g3").gain(mousey)
