3) Modifiers – Time

FonctionDéfinition détaillée
slow(n).Multiplie la durée du pattern par n cycles
fast(n).Divise la durée du pattern par n
rev().Inverse l’ordre temporel des événements
early(n).Avance les événements de n cycles
late(n).Retarde les événements de n cycles
iter(n).Fait tourner le pattern sur n itérations
ply(n).Duplique chaque événement n fois
euclid(p, s).Génère un pattern euclidien p pas sur s
segment(n).Découpe le cycle en n segments égaux
chunk(n, fn).Applique fn par blocs de n cycles

4) Modifiers – Random

FonctionDéfinition détaillée
choose(list).Sélection uniforme aléatoire par événement
wchoose(list).Sélection pondérée
degrade().Supprime ~50% des événements
degradeBy(p).Supprime avec une probabilité p
undegrade().Inverse l’état degrade
sometimes(fn).Applique fn avec probabilité moyenne
often(fn).Applique fn avec probabilité élevée
rarely(fn).Applique fn avec probabilité faible
rand().Valeur aléatoire continue (0–1)
irand(n).Entier aléatoire de 0 à n-1

5) Modifiers – Conditional

FonctionDéfinition détaillée
firstOf(n).Actif pendant les n premiers cycles
lastOf(n).Actif pendant les n derniers cycles
when(cond, fn).Applique fn si cond est vrai
mask(pattern).Filtre les événements selon un masque
pick(n).Sélectionne l’événement d’index n
invert().Inverse un masque logique
hush().Muting global du pattern
