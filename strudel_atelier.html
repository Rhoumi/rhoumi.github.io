Fonctions mélodiques

Fonction Définition Variables / valeurs possibles
.note("") Fonction basique de définitions de notes liste de notes (Numéro MIDI, note en chiffrage américain "c", nombre de 0 à 13, note absolue "c2"
.scale(name) Transforme des degrés en notes selon une échelle donnée. Définit aussi l’échelle par défaut pour d’autres opérations tonales. string de forme ":" comme "C:major", "A2:minor", "F#:dorian"
.transpose(semitones) Transpose toutes les notes d’un pattern par un nombre de demi‑tons. entier ou pattern d’entiers (ex: 2, -3, "<0 2 -1>")
.scaleTranspose(steps) Transpose les notes *à l’intérieur de l’échelle* d’un certain nombre de degrés. entier ou pattern d’entiers
.voicings(range?) Convertit des symboles d’accord en voicings (positions de notes), avec un voice leading « smooth ». optionnel : range / dictionnaire voicing (ex: 'lefthand', 'root:g2')
.rootNotes(octave = 2) Transforme des symboles d’accord en notes fondamentales dans une octave donnée. octave (entier), pattern de symboles d’accord
