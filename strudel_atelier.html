Guide de référence pour Strudel.cc

1. Mini-Notation : Syntaxe de base et motifs

Caractère Fonction Exemple Description
< > Séquence <a b c> Joue a, puis b, puis c
[ ] Empilement [bd hh] Joue bd et hh simultanément
/ Division bd/2 Joue bd deux fois plus lentement
* Multiplication sd*4 Joue sd quatre fois plus vite
! Sélection aléatoire sd!3 Sélectionne 3 événements du motif aléatoirement
~ Pause / Silence <a ~ b> Joue a, puis pause, puis b
& Parallèle a&b Joue a et b simultanément (comme Stack mais plus flexible)
, Choix aléatoire <a,b,c> Choisit aléatoirement entre a, b ou c
@ Décalage temporel bd@2 Décale le motif de 2 unités vers l'avant
s() Jouer un sample s("bd") Joue le sample de grosse caisse
n() Jouer des notes n("c4") Joue la note Do4

2. Samples et instruments

Strudel dispose d'une série de samples intégrés et peut également charger des paquets externes.

Raccourci Sample Description
bd kick Grosse caisse
sd snare Caisse claire
cp clap Claquement de mains
hh hi-hat Charleston (fermé)
oh open hi-hat Charleston (ouvert)
cr crash cymbal Cymbale crash
ride ride cymbal Cymbale ride
lt low tom Tom basse
mt mid tom Tom medium
ht high tom Tom aigu

Instruments généraux :

  • s("synth") : Un synthétiseur analogique classique
  • s("fm") : Un synthétiseur FM
  • s("superfm") : Un synthétiseur FM avec oscillateur de type SuperSaw
  • s("string") : Génération sonore type cordes
  • s("am") : Un synthétiseur AM
  • s("sine") : Un oscillateur simple à onde sinusoïdale

Sons General MIDI (GM) :

  • s("gm_acoustic_grand_piano") : Piano acoustique
  • s("gm_acoustic_guitar_steel") : Guitare acoustique à cordes d'acier
  • s("gm_electric_bass_finger") : Basse électrique
  • s("gm_violin") : Violon
  • ... et bien d'autres, accessibles via gm_[nom_instrument]

3. Effets et modificateurs de paramètres

Les effets sont ajoutés à un motif avec la syntaxe par point.

Fonction Paramètre Exemple Description
gain() 0.0 - 1.0 .gain(0.5) Règle le volume
room() 0.0 - 1.0 .room(.7) Ajoute de la réverbération
delay() Temps en cycles .delay(.25) Ajoute un écho
pan() -1.0 - 1.0 .pan(-1) Position dans le panoramique stéréo
lpf() Fréquence (Hz) .lpf(1000) Filtre passe-bas
hpf() Fréquence (Hz) .hpf(8000) Filtre passe-haut
shape() 0.0 - 1.0 .shape(.3) Effet de distorsion
crush() 1 - 16 .crush(8) Bitcrusher
phaser() Fréquence .phaser(4) Effet phaser
speed() Valeur .speed(.5) Modifie la vitesse de lecture des samples
note() Gamme .note().scale("c:minor") Joue les notes comme mélodie depuis la gamme
chord() Nom d'accord .chord("Cmaj7") Joue un accord

4. Temps, gammes et fonctions génératives

Fonction Exemple Description
setcps() setcps(0.75) Définit la vitesse en Cycles par Seconde
setbpm() setbpm(120) Définit la vitesse en Battements par Minute
rand() rand.range(0,1) Génère un nombre aléatoire
perlin() perlin.range(.6, .9) Génère des valeurs aléatoires fluides (bruit de Perlin)
sine() sine.range(1, 10) Génère une onde sinusoïdale
scale() scale("c:minor") Convertit les notes dans une gamme spécifique
pick() pick([a,b,c]) Choisit un motif dans une liste
pickRestart() pickRestart([a,b]) Choisit un motif et redémarre depuis le début quand il se termine
